Арифметические операции
Работают с числовыми типами (int, float, double и т. д.).

Операция	Символ	Пример
Сложение	+	a + b
Вычитание	-	a - b
Умножение	*	a * b
Деление	/	a / b (целочисленное деление для int)
Остаток от деления	%	a % b
✅ Примеры:

csharp
Копировать
Редактировать
int a = 10;
int b = 3;

Debug.Log(a + b); // 13
Debug.Log(a - b); // 7
Debug.Log(a * b); // 30
Debug.Log(a / b); // 3 (целочисленное деление)
Debug.Log((float)a / b); // 3.333 (дробное деление)
Debug.Log(a % b); // 1 (остаток от деления)

Инкремент (++) и декремент (--)

csharp
Копировать
Редактировать
int x = 5;
x++; // x = x + 1
Debug.Log(x); // 6

int y = 5;
y--; // y = y - 1
Debug.Log(y); // 4
