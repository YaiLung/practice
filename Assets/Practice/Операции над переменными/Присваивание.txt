Операции присваивания
Позволяют изменять переменные без повторного их написания.

Операция	Пример
+=	a += 5; (аналог a = a + 5;)
-=	a -= 3; (аналог a = a - 3;)
*=	a *= 2; (аналог a = a * 2;)
/=	a /= 2; (аналог a = a / 2;)
%=	a %= 2; (аналог a = a % 2;)
✅ Пример:

csharp
Копировать
Редактировать
int a = 10;
a += 5; // a = 15
a *= 2; // a = 30
a /= 3; // a = 10
