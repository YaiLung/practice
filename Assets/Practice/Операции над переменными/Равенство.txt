Операции равенства и неравенства
Используются для сравнения значений простых типов (int, float, double, char, bool, enum и т. д.).

Операция	Символ	Пример
Равно	==	a == b
Не равно	!=	a != b
✅ Пример с числами:

csharp
Копировать
Редактировать
int a = 10;
int b = 10;
Debug.Log(a == b); // true
Debug.Log(a != b); // false
