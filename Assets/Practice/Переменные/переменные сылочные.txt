Основные ссылочные типы
Классы (class)
Классы всегда создаются в куче (heap) и передаются по ссылке.

csharp
Копировать
Редактировать
public class Player

Массивы (array)
Массивы в C# тоже являются ссылочным типом.

csharp
Копировать
Редактировать
int[] numbers1 = { 1, 2, 3 };

Строки (string)
Хотя string — ссылочный тип, он ведёт себя как значимый тип, потому что является неизменяемым (immutable).

csharp
Копировать
Редактировать
string str1 = "Hello";
string str2 = str1;
