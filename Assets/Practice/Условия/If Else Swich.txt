Условия в C# (if, else, switch)
В C# условия используются для принятия решений в коде. Основные конструкции: if, else, else if, switch.

Условный оператор if
Позволяет выполнить код только при выполнении условия.

✅ Пример:

csharp
Копировать
Редактировать
int health = 50;
if (health > 0)
{
    Debug.Log("Игрок жив!");
}

Оператор if-else
Позволяет выполнять разные действия в зависимости от условия.

✅ Пример:

csharp
Копировать
Редактировать
int health = 0;
if (health > 0)
{
    Debug.Log("Игрок жив!");
}
else
{
    Debug.Log("Игрок мертв!");
}

Оператор switch-case
Используется для проверки множества значений одной переменной.

✅ Пример:

csharp
Копировать
Редактировать
int day = 3;

switch (day)
{
    case 1:
        Debug.Log("Понедельник");
        break;
    case 2:
        Debug.Log("Вторник");
        break;
    case 3:
        Debug.Log("Среда");
        break;
    default:
        Debug.Log("Выходной!");
        break;
}
